# create-stratawp

Create modern WordPress themes with StrataWP using one command.

## Usage

```bash
npx create-stratawp my-theme
```

This will guide you through creating a new WordPress theme with:
- TypeScript & React support
- Vite for lightning-fast HMR
- Block Theme (FSE) architecture
- Modern tooling and best practices

## What's Inside

This package is a thin wrapper that installs and runs [@stratawp/cli](https://www.npmjs.com/package/@stratawp/cli).

For more information, visit the [StrataWP repository](https://github.com/JonImmsWordpressDev/StrataWP).

## Requirements

- Node.js 18+
- WordPress 6.7+
- PHP 8.1+
